JAVA

1. Literals
Literals are constant values written directly in the code. They represent fixed data like numbers,characters, or strings.

2. Keywords
Keywords are reserved words in Java that have a special meaning. You cannot use them as variable or method names.
Examples: class, int, public, static, return, if

3. Identifiers
Identifiers are the names used for variables, methods, classes , etc. They help in identifying different elements of the program.
Example: int age; // 'age' is an identifier

4. Arrays
An array is a container that holds multiple values of the same type.
It helps in storing and accessing data using index positions.
Examples: int[] marks = {90, 80, 70};

5. Methods
A method is a block of code that performs a specific task.
It helps in code reuse and modular programming.
Examples: static void greet() { System.out.println("Hello!"); }

6. Operations
Operators are symbols that perform operations on variables and values.
Types of Operators:
Arithmetic: + - * / %
Relational: == != > < >= <=
Logical: && || !
Assignment: = += -=
Unary: ++ --
Ternary: condition ? true : false
Bitwise: & | ^ << >>

7. Looping Techniques
Loops are used to execute a block of code repeatedly.
Types:
For Loop – Runs for a fixed number of times.
While Loop – Runs while a condition is true.
Do-While Loop – Runs at least once, then checks the condition.
Enhanced For Loop – Used to loop through arrays or collections.
Examples: for(int i = 1; i <= 5; i++) { System.out.println(i); }
